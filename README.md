# API LOAN(Emprestimos)

## Overview
Desafio SRM

## Stack

- Java 17
- Spring Boot
- Maven
- Postgres
- Docker

## Executar API

1. Clonar repositório - https://github.com/bob-burgao/api-loan.git

2. Baixar dependências - `mvn clean install -DskipTests`

3. Executar o comando docker - `docker-compose up -d`
